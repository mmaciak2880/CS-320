Over the course of this project it was important to ensure that my programs were functional and secure. To make sure everything was functional I wrote JUnit tests. To check that the program is functioning as required. By making variables and functions private as needed I was also able to ensure that the programs were secure. For this project I was able to interrupt the users needs through the user requirements document. My approach for designing the programs was simple and straight forward. I took the user requirements, broke them up into sections of what could be done in what order, and then built one section at a time. Until all the sections were completed.
